Build Instructions

This chapter describes all the JAR files you need to use sample codes.

DFS Client API

Following JARs need to be included in classpath to use sample codes related to DFS CLient APIs.

  • $INSTALL_HOME/services/hadoop-2.7.1/share/hadoop/common/hadoop-custom-compiled-server.jar
  • $INSTALL_HOME/services/hadoop-2.7.1/share/hadoop/common/lib/commons-logging-1.1.1.jar
  • $INSTALL_HOME/services/hadoop-2.7.1/share/hadoop/common/lib/log4j-1.2.17.jar
  • $INSTALL_HOME/services/hadoop-2.7.1/share/hadoop/common/lib/slf4j-api-1.6.1.jar
  • $INSTALL_HOME/services/hadoop-2.7.1/share/hadoop/common/lib/slf4j-log4j12-1.6.1.jar
  • $INSTALL_HOME/services/hadoop-2.7.1/share/hadoop/common/lib/protobuf-java-2.5.0.jar
  • $INSTALL_HOME/services/hadoop-2.7.1/share/hadoop/common/lib/guava-11.0.2.jar
  • $INSTALL_HOME/services/hadoop-2.7.1/share/hadoop/common/lib/commons-lang-2.5.jar
  • $INSTALL_HOME/services/hadoop-2.7.1/share/hadoop/common/lib/commons-configuration-1.6.jar
  • $INSTALL_HOME/services/hadoop-2.7.1/share/hadoop/common/lib/commons-cli-1.2.jar
  • $INSTALL_HOME/services/hadoop-2.7.1/share/hadoop/common/lib/commons-io-2.1.jar

S3 Compatible REST API

Following JARs need to be included in classpath to use sample codes related to S3 Compatible REST API.

  • httpcore-4.2.2.jar
  • httpclient-4.2.2.jar
  • httpclient-cache-4.2.2.jar
  • httpmime-4.2.2.jar

Writing MapReduce Jobs for AdHoc Querying / On Ingest Parser

Following JARs need to be included in classpath to use sample codes.

  • $INSTALL_HOME/hadoop-2.7.1/share/hadoop/common/queryio-plugins.jar
