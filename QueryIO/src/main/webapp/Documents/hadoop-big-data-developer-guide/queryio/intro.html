Introduction

QueryIO is a Hadoop based Big Data Query and Analytics solution.

This guide describes how to use the QueryIO's Developers API. You can use the QueryIO APIs to develop applications to store, share and manage data from anywhere on the Internet.

This guide explains all the supported interfaces and their APIs that you may use to perform read/write operations on QueryIO. Sample codes are also provided for this purpose.

Supported APIs are :

  • DFS Client API
  • S3 Compatible REST API
  • WEBHDFS API

This document is not a programming guide. It is assumed that you have the necessary skills for writing code in the appropriate language.

You can find the complete source code for demo programs in $INSTALL_DIR/demo/ directory.
