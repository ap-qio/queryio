GET Hadoop SQL List

Description

GET Hadoop SQL List operation of QueryIO server will fetch all Hadoop SQL Queries from the database.

Requests

Syntax

  • GET /queryio/hadoopsql/ HTTP/1.1
Host: QueryIO.com
authorization: Token
  • authorization: Token for authorization.

Examples

Sample Request

The following request will retrieve all the Hadoop SQL Queries.

  • GET /queryio/hadoopsql/ HTTP/1.1
Host: QueryIO
authorization: iffo6l9hel2hfmbj2384joljgh9mqga58gb9if9593ucli9ke5s2e3854shhcmmm
