DELETE Bucket

Description

DELETE Bucket operation of QueryIO server will delete an existing bucket. Bucket should be empty before it can be deleted.

Requests

Syntax

  • DELETE /queryio/<BucketName> HTTP/1.1
Host: QueryIO.com
authorization: <Token>
  • <BucketName>: Name of the bucket to be deleted.
  • authorization: Token for authorization.

Examples

Sample Request

The following request deletes a bucket named DIR1.

  • DELETE /queryio/DIR1 HTTP/1.1
Host: QueryIO.com
authorization: iffo6l9hel2hfmbj2384joljgh9mqga58gb9if9593ucli9ke5s2e3854shhcmmm
