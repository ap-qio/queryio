GET Object ACL

Description

GET object ACL operation of QueryIO server will an object and its related ACL settings.

Requests

Syntax

  • GET /queryio/<BucketName>/<ObjectName>?acl HTTP/1.1
Host: QueryIO.com
authorization: Token
  • <BucketName>: Name of the bucket which consists object.
  • <ObjectName>: Name of the object to be fetched.
  • authorization: Token for authorization.

Examples

Sample Request

The following request fetches an object.

  • GET /queryio/DIR1/demo.txt?acl HTTP/1.1
Host: QueryIO.com
authorization: iffo6l9hel2hfmbj2384joljgh9mqga58gb9if9593ucli9ke5s2e3854shhcmmm
