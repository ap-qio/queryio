PUT Bucket ACL

Description

PUT Bucket ACL operation of QueryIO server will create bucket set ACL properties on bucket.

Requests

Syntax

  • PUT /queryio/<BucketName>?acl HTTP/1.1
Host: QueryIO.com
authorization: Token
  • <BucketName>: Name of the bucket to be created.
  • authorization: Token for authorization.

ACL Headers

QueryIO server uses following ACL headers:

Header Name Description Type Required
owner

Set owner of the bucket.

 String No
group

Set group of the bucket.

 String No
permission

Permissions to be granted on bucket in octal format.

 OctalFormat No

Examples

Sample Request

The following request will set ACL settings on bucket.

  • PUT /queryio/DIR1?acl HTTP/1.1
Host: QueryIO.com
authorization: iffo6l9hel2hfmbj2384joljgh9mqga58gb9if9593ucli9ke5s2e3854shhcmmm
permission: 777
