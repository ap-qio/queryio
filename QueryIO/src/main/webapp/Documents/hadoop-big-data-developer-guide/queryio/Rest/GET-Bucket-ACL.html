GET Bucket ACL

Description

GET bucket ACL operation of QueryIO server will an bucket contents and its related ACL settings.

Requests

Syntax

  • PUT /queryio/<BucketName>?acl HTTP/1.1
Host: QueryIO.com
authorization: Token
  • <BucketName>: Name of the bucket to be fetched.
  • authorization: Token for authorization.

Examples

Sample Request 

  • PUT /queryio/DIR1?acl HTTP/1.1
Host: QueryIO.com
authorization: iffo6l9hel2hfmbj2384joljgh9mqga58gb9if9593ucli9ke5s2e3854shhcmmm
