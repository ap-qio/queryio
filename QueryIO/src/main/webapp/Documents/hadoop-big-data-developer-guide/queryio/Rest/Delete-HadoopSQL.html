DELETE Hadoop SQL

Description

DELETE Hadoop SQL operation of QueryIO server will delete given Hadoop SQL query from the database.

Requests

Syntax

  • DELETE /queryio/hadoopsql/ HTTP/1.1
Host: QueryIO.com
authorization: <Token>
  • authorization: Token for authorization.

Request Headers

The Delete Hadoop SQL operation in QueryIO Server uses following request headers:

Header Name Description Type Default Required
hadoopsql-id

Id for Hadoop SQL query to delete

 String None Yes

Examples

Sample Request

The following request will delete a Hadoop SQL query with id "Query1".

  • DELETE /queryio/hadoopsql/ HTTP/1.1
Host: QueryIO
authorization: iffo6l9hel2hfmbj2384joljgh9mqga58gb9if9593ucli9ke5s2e3854shhcmmm
hadoopsql-id : Query1

Sample Response

  • <xml version="1.0" encoding="UTF-8">
<DeleteHadoopSQL>
	<Status><true></Status>
</DeleteHadoopSQL>
