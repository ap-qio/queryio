PUT Object ACL

Description

PUT Object ACL operation of QueryIO server will create object and set ACL properties on object.

Requests

Syntax

  • PUT /queryio/<BucketName>/<ObjectName>?acl HTTP/1.1
Host: QueryIO.com
authorization: <Token>
  • <BucketName>: Name of the bucket which consist of object.
  • <ObjectName>: Name of the object.
  • authorization: Token for authorization.

ACL Headers

QueryIO server uses following ACL headers:

Header Name Description Type Required
owner

Set owner of the object.

 String No
group

Set group of the object.

 String No
permission

Permissions to be granted on object in octal format.

 OctalFormat No

Examples

Sample Request

The following request will set ACL settings on object.

  • PUT /queryio/DIR1/demo.txt?acl HTTP/1.1
Host: QueryIO.com
authorization: iffo6l9hel2hfmbj2384joljgh9mqga58gb9if9593ucli9ke5s2e3854shhcmmm
permission: 700
