Update Hadoop SQL

Description

Update Hadoop SQL operation of QueryIO server will add given Hadoop SQL query to the database.

Requests

Syntax

  • GET /queryio/hadoopsql/update/ HTTP/1.1
Host: QueryIO.com
authorization: <Token>
  • authorization: Token for authorization.

Request Parameters

The Update Hadoop SQL operation in QueryIO Server uses following request parameters:

Parameter Name Description Return Type Default Required
hadoopsql-properties

Query properties JSON

 String None Yes

Examples

Sample Request

The following request will fetch objects from NewBucket.

  • GET /hadoopsql/update/ HTTP/1.1
Host: QueryIO
Authorization: iffo6l9hel2hfmbj2384joljgh9mqga58gb9if9593ucli9ke5s2e3854shhcmmm
hadoopsql-properties : {"sqlQuery":"SELECT *  FROM TAGS_NAMENODE1","chartDetail":{},"selectedOrderBy":[],"selectedTable":["TAGS_NAMENODE1"],"queryHeader":{"header":{"title":""}},"selectedWhere":{},"queryFooter":{"footer":{"title":""}},"groupHeader":{},"queryDesc":"","groupFooter":{},"colDetail":{},"colHeaderDetail":{},"selectedGroupBy":[],"selectedColumn":{"*":{}},"queryId":"Query1"}

Sample Response

  • <xml version="1.0" encoding="UTF-8">
<UpdateHadoopSQL>
	<Status><true></Status>
</UpdateHadoopSQL>
