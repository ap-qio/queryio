Introduction

QueryIO is a Hadoop based Big Data Query and Analytics solution.

Hands On Tutorial will help you get started with using QueryIO. This guide is useful for those who want to dive straight in but need to know where to start.

The guide is designed to give the user a conceptual overview of all of the activities needed to setup a QueryIO cluster. This document touches briefly many important and complex concepts and does not provide a detailed explanation of any one topic since the intent is to present the material in the most convenient and practical manner.

The goal is simply to help the user become familiar enough with the product to successfully setup cluster, perform data operations on cluster like uploading or querying data, tagging data and adding MapReduce jobs for data processing.
