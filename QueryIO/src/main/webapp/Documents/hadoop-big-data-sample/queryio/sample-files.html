Samples directory: Sample files for Hive Data Definition

$INSTALL_HOME/demo/samples directory consist of sample file for each supported data type by Hive data definition.

Sample files for following data types are available :

  • Apache access log
  • CSV
  • IIS log
  • JSON
  • Key Value
  • Log4j
  • Mbox
  • XML
