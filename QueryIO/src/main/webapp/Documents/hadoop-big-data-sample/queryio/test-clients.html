Data Management Samples

$INSTALL_HOME/demo/TestClients.zip consist of various sample programs in various APIs, supported by QueryIO.

Sample programs can be used for data management on QueryIO cluster through following supported APIs:

  • DFS Client API
  • WEBHDFS API
  • S3 Compatible REST API

Using sample programs, you can add, edit, delete, retrieve files, directories and queries from QueryIO.
