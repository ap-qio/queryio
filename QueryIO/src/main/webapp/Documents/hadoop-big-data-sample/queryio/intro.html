Sample Apps and Data

QueryIO is a Hadoop based Big Data Query and Analytics solution. QueryIO ships with a number of sample applications and sample data to expedite the learning and evaluation process. These apps and data can be used to explore major features of QueryIO. Please go through QueryIO Tutorials to learn how to use these apps and data.

Following sample apps and data are included for various QueryIO features :

  • Data.zip : It contains sample files which can be uploaded to QueryIO server.
  • LogParserMRJob.jar : A sample MapReduce job for LOG files.
  • TestClients.zip : It contains all the sample programs for data processing in various APIs supported by QueryIO.
  • NCDC weather data parser: An AdHoc mapreduce job to parse the weather data accumulated by NCDC.
  • Wikipedia xml database dump parser: An On Ingest data tagger to parser wikipedia database.

This document will help you to use these sample apps and data.
