Starting QueryIO Server through Command Prompt

This chapter covers information relating to starting QueryIO server manually and accessing QueryIO server UI.

Startup Instructions

Use following steps to manually start QueryIO server:

  • Start terminal.
  • Navigate to tomcat/bin folder in installation directory.
    • cd $INSTALL_HOME/tomcat/bin
  • Execute start_queryio script file to start QueryIO server.
    • To start server:
        • sh start_queryio.sh

        OR

        ./start_queryio.sh

Once QueryIO server starts up, you can use your browser to access it.

Accessing QueryIO server

You can access QueryIO server in your browser using the hostname/IP and UI Server port which you have mentioned while installing. Default port is 5678.

URL: http://$HOSTNAME:$PORT/queryio/
