Q: Installer is not working fine. I am getting ELF bad interpreter error on trying to install it on Linux.

You need to install "ld-linux.so.2". You can install it using following command:

  • yum install ld-linux.so.2

Q: Does QueryIO support windows platform?

No, QueryIO does not support windows platform.


Q: Does QueryIO require Administrative privileges for Installation?

No, you don't need administrative privileges for installation. But you must have access to destination location where software is installed.


Q: On completion of installation the QueryUI server is not starting. What should I do?

Try starting the QueryIO server manually. Goto $INSTALL_HOME/tomcat/bin and run following executable file:

  • For Linux/Mac OS: ./start_queryio.sh

Q: In remote installation setup, installer is unable to SSH to remote machine. What should I do?

Please check that SSH service is enabled on remote machine and credentials provided are correct.


Q: What is "First time user registration"?

This screen appears when you visit QueryIO UI for the first time. Server needs atleast one admin user.


Q: After running the server successfully, QueryIO UI is not properly displayed in my browser.

The reason might be outdated version of your browser. Please update your browser. See System Requirements for more details.
