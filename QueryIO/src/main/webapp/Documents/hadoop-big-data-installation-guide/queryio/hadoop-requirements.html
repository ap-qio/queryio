System Requirements

This chapter covers information relating to the minimum system requirements and supported products.

Minimum System Requirements

Minimum system requirements for installing QueryIO, a Big Data Analytics are listed below:

System Parameter

Requirement

Operating System

Linux, Mac OS X

RAM

512 MB

CPU

1 GHz

Disk Space

500 MB

JRE

JRE version.1.7

Note: The preferred screen area or resolution for the QueryIO server is 1024 x 768 pixels.

Supported Products

QueryIO supports:

Products Category

Products Supported

Operating Systems

Linux x86, Mac OS X

Browsers

Mozilla Firefox v.10.0.0 or above , Safari v.5.0.0 or above, Google Chrome v.23.0.0.0 or above, Internet Explorer v.8 or above

Databases

All databases that have JDBC drivers and support JDBC specifications
