Getting Help and Support

This chapter provides overview of how to access the documentation provided with QueryIO and how to access Help while using QueryIO.

Using the Documentation

Help can be invoked from anywhere in the application by clicking 'Help' link at the top right corner of the window.

Alternatively, you can access the documentation at following locations:

QueryIO Tutorial

http://$HOSTNAME:$INSTALL_PORT/queryio/docs/hadoop-big-data-tutorial/index.html

Admin Guide

http://$HOSTNAME:$INSTALL_PORT/queryio/docs/hadoop-big-data-admin-guide/index.html

Developer Guide

http://$HOSTNAME:$INSTALL_PORT/queryio/docs/hadoop-big-data-developer-guide/index.html

Getting Product Support

If you need additional assistance with our products, please contact QueryIO at the email address specific to your query:

Sales: sales@queryio.com

Support: support@queryio.com
