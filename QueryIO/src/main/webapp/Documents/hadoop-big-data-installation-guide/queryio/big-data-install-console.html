QueryIO: Big Data Analytics - Console Based Installation

In this chapter

This chapter explains about console based QueryIO Installation. When you install QueryIO, a number of components are installed, including Hadoop, databases, libraries, services, agents, etc.

Installation

Open a terminal window and change the current directory to the location where you have saved QueryIO installer (e.g. QueryIO_v4_5_linux.sh etc). Now type the following command and press the Enter key.

  • sh QueryIO_v4_5_linux.sh -c
  • This will launch the QueryIO installer in console mode.

Local Installation

  • Press <ENTER> or "o" to install QueryIO
  • Press "c" to cancel installation.

  • License Agreement:
    • Press "Enter" to accept and continue installation.

  • QueryIO User Registration Details:
    • Give details to make first admin account in QueryIO. Remember this username and password for logging in after server starts.

  • Select Destination Directory
    • Enter the installation directory.
    • Press <ENTER> to install at default destination directory or enter new installation location and press <ENTER>
    • JRE Home: Root folder for JRE on local machine.

  • Choose Link Location
    • There are many options available for creating application links. A message asking "Create symlinks? Yes [y, Enter], No [n]" will be displayed.
    • If you want to create link type "y" and press the <ENTER> key to continue or if you don't want to create link, press "n".
    • If you have selected yes, next step will ask for the location to create the links. Please provide the location where you want links to be created and press the <ENTER> key to continue.




  • Configure QueryIO Server Details
    • Specify the QueryIO Server port settings. The QueryIO Services requires following ports:
      • Startup and Shutdown port: Ports on which QueryIO server will work.
      • System Database port: Ports for QueryIO database.
    • Press <ENTER> to use default port or enter new port and press <ENTER>
    • We recommend using the default ports provided by QueryIO.




  • Automatic Hadoop Cluster Setup
    • This will install hadoop cluster on chosen machine and make all nodes automatically on that machine (on default ports).
    • Press "1" to proceed with automatic cluster setup.
    • Press "2" to complete setup without setting up cluster (complete cluster setup manually after server starts).
  • Filling Hadoop Cluster Setup Form
    • Setup Hadoop cluster on local machine : Select this if you want to setup cluster on local machine.
    • Setup Hadoop cluster on remote machine : Select this if you want to setup cluster on remote machine.
    • SSH Details (If you choose remote machine)
      • HostName/IP: IP address or name of the host to be added.
      • Username: Username of the host.
      • Password: Password for account whose username was provided.
  • Setting up properties for continuing installation
    • Setup fixes issues like, no mapping for hostname found automatically (if required).
    • Please provide machine Admin Username and Password to run commands required to fix these issues.
    • If settings are updated successfully, confirmation is shown.




  • Press <ENTER> to continue installation.

  • Installation Complete
    • After installation is complete, QueryIO will ask to start server automatically. Press <ENTER> to Start QueryIO server or press "n" to start QueryIO server manually.
    • Start Automatically on System Reboot: Press <ENTER> to start QueryIO server every time system is restarted. Press <n> to cancel option.
    • On successful installation a message "Finished installing QueryIO" is displayed.
