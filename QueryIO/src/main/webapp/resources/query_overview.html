Analytics

Analytics section helps you in managing and running queries on data to generate reports and creative visualizations.

Query Manager

Query Manager is where you add, modify, view and delete queries for your reports.

Begin by clicking on Add button in Query Manager.

Query Designer

Lets you design your analytics queries with a simple yet powerful interface.

Go to Query Designer.

Report viewer

Report viewer shows you the results of running a query in HTML compatible format in your browser. You can also export the results to common file formats.

Click here to go to the Report Viewer.

Spreadsheet Viewer

Spreadsheet viewer is a tool to analyze the results with spreadsheet capabilities.

Click here to open the results of a query in Spreadsheet format.
